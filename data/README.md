# Input data

Input files are not distributed with this repository.

The analysis expects:

- an equal-length processed marker alignment in FASTA format; and
- a binary virulence-response table containing isolate identifiers, optional published pathotype labels, and host-differential responses encoded as `R` or `S`.

The processed marker alignment used in the associated study was generated by the original study team and requires authorization from the original data custodians for redistribution.
