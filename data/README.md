# Data files

Input data are not bundled with this repository unless permitted by the relevant data source.

The pipeline expects:

1. An aligned FASTA file containing *Colletotrichum sublineola* isolate sequences.
2. A virulence matrix containing `isolate_id` and host differential response calls encoded as `R` or `S`.

The full analytical output workbook used in the manuscript should be provided separately as supplementary data.
